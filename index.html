WebDGap

http://mikethedj4.github.io/WebDGap/

Creating cross-platform web applications can no longer be a headache anymore thanks to WebDGap! As a matter of fact. WebDGap is the first web application that enabled users to create mac applications in the browser! Guess what? It's free and open source!

WebDGap has been featured on websites such as Infotonics Media, TekZoom, Somewhere.com (yep that's a real domain name) and Windows Report.

kodeWeave

http://mikethedj4.github.io/kodeWeave/

kodeWeave is a free and open source HTML/CSS/JavaScript front-end live development playground. Similar to other coding playgrounds like Codepen and JSFiddle. It essentually serves the same purpose. The big difference is that in addition to using it as a web app, you can download it as a standalone app for Windows, Mac, Linux, ChromeOS and Android. In addition it also comes equipped with the use of preprocessors and about every framework you can name off the top of your head.

kodeWeave has been featured on websites such as Web Designer Depot and Web Designity.

ArabEngo

http://mikethedj4.github.io/ArabEngo/

ArabEngo is a free and open source language learning application in which native English speakers can learn Arabic! I also released a fork of ArabEngo called langApp in which you all can play with and use it to make your own open source language learning application as well.

CamDesk

http://mikethedj4.github.io/CamDesk/

CamDesk was one of the first applications I created strictly for productivity. While it's premise was to be a simple home surveillance application. It's also been used by many others for demonstrational purposes via screencasting.

The first release of CamDesk was back in 2010 when I was 18. Now with over 1,000,000 accumulating from sites all across the web (with $0 spent on marketing, as it's a free and open source application). CamDesk has been featured on popular websites like OMGUbuntu and MakeUseOf.
